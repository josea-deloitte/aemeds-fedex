# CTA Banner

A full-bleed FedEx purple strip with a headline, one supporting sentence, and a single call to action (e.g. the FedEx Rewards sign-up banner).

## Authoring

Create a table named `CTA Banner` with a single cell:

| CTA Banner |
| --- |
| ## Get rewarded for shipping<br>Join FedEx Rewards for free and earn on eligible shipments and purchases.<br>**[Join FedEx Rewards](/en-us/rewards.html)** |

| Element | Renders as |
| --- | --- |
| Heading 2 | White headline, centered |
| Paragraph | White supporting copy (keep to one sentence) |
| **Bold link** | Orange button |
| Plain link | White underlined text link |

## Good to know

- The purple background stretches edge to edge regardless of where the block sits; the content stays centered.
- Put the banner in its own section. One CTA per banner — if you need two actions, use two banners or a different block.
